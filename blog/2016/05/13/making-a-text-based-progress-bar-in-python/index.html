Making a Text Based Progress Bar in Python

13 May 2016 Python • বাংলা • কোডিং • coding • code • কোড • কনসোল • console

ছোটবেলায় যখন কনসোলে কিছু একট কাজ করতে দিতাম - দেখা যেত একটা প্রোগ্রেস বারও আছে। সেটা আবার এক যায়গায় আস্তে আস্তে বাড়তে থাকে। মনে হতো, আরে!কনসোলে এই যে প্রোগ্রেসবারটা দেখি, সেটা কিভাবে কাজ করে? এনিমেশনটা হয় কিভাবে?! ভাবলাম এখন যখন জানি, সবাইকে দেখাই কেমন করে কাজ করে! কোডটা দিলাম, কিন্তু ছবি আকারে - করতে চাইলে কোডটা নিজেই লিখে করো :D

কোডের আউটপুটটা দেখতে এমন হবে:

লক্ষ্য করার যে অংশটা, সেটা হলো \r. এটার কারণেই একযায়গায় বার বার লেখা হচ্ছে। ‌\r মানে হচ্ছে ক্যারিজ রিটার্ন। আর ফ্লাশিংয়ের অংশটা এখান থেকে বুঝতে পারো।
